## 执行命令

### Build Setup

```bash
# 安装依赖包
$ yarn install

# 在本地执行命令进行热重载 localhost:3031
$ yarn dev

# 该执行命令将项目生产并启动服务器构建
$ yarn build
$ yarn start

# 生成静态项目
$ yarn generate

# 生成并分析报告
$ yarn build -a
```
